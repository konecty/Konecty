## Format

Each ADR follows the standard format:
- **Status**: Accepted, Proposed, Rejected, Deprecated, Superseded
- **Context**: Situation that led to the decision
- **Decision**: Decision made
- **Implementation Details**: How it was implemented
- **Consequences**: Positive, negative impacts and mitigated risks
- **Alternatives Considered**: Other options evaluated
- **References**: Links to related code
